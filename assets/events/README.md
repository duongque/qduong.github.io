# Event preview covers

Dépose ici les images de couverture pour les sous-encadrés "Events & Training". Le portfolio les affiche en miniature à gauche du titre, dans chaque carte d'événement.

## Fichiers attendus

| Clé HTML       | Fichier         | Contenu suggéré                                                  |
|----------------|-----------------|------------------------------------------------------------------|
| `cas.png`      | `cas.png`       | Logo / affiche CAS (CERN Accelerator School)                     |
| `ifast.png`    | `ifast.png`     | Logo / capture du projet i-FAST                                  |

## Format

- **PNG ou JPG** (transparent ou opaque)
- **Format carré ou 4:3 recommandé** (l'image est crop centrée à 64×64 dans le rendu)
- Min. 128×128 px pour rester nette

## Fallback automatique

Si l'image n'existe pas encore, le sous-encadré affiche un placeholder gris hachuré avec une flèche `↗`. Tu peux donc déposer les images au fil de l'eau.
